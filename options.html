open-os Extension Options

Small open source extension for Edge, Firefox and Chrome to quickly access to your favorite local AI LLM assistan while browsing.

This beta is configured for Ollama models.

Simply click on the extension icon and start chatting with your virtual assistant. Right-click on the extension icon to open in a new tab.

Extension has been developed and is maintained by @luishg with the help of ChatGPT and Github Copilot.


Make sure you have installed Ollama and it is running
  • Download Ollama: https://ollama.com/
  • Llama-3 Installation tutorial: https://www.youtube.com/watch?v=7ujZ1N4Pmz8

Not getting responses? How to expose the Ollama server.

By default, Ollama allows cross origin requests from 127.0.0.1 and 0.0.0.0.

To support more origins, you can use the OLLAMA_ORIGINS environment variable: https://github.com/ollama/ollama/blob/main/docs/faq.md

For more info visit open-os.com
